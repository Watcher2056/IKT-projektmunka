# IKT projektmunka


## Megrendelői igények

Egy MMS (Music Streaming Service) weboldal a népszerű MMS weboldalak alapfunkcióival, késő 90-es, korai 2000-es stiluselemekkel.

### Főbb funkciók:

- Saját könyvtár, Ajánlott/Újdonságok és Podcast menüpontok
- Felhasználói profil
- Media control kezelőfül
- Regisztrációs illetve bejelentkezési portál
- Reszponziv layout

###	Funkcionalitás részletezése

Zenei streaming élmény nyújtása.Lejátszási listák egyszerű létrhozása illetve kezelése a felhasználó által.  Támogatja a felhasználói hitelesítést.Reszponzív tervezés asztali webkeresőkre.
